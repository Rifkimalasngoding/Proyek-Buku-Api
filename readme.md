# Proyek API Buku Sederhana

Proyek ini adalah contoh aplikasi web sederhana yang mengimplementasikan RESTful API menggunakan Node.js (Express) dan antarmuka Vanilla HTML/CSS/JS.

## Cara Menjalankan

1. Pastikan kamu sudah menginstal [Node.js](https://nodejs.org/).
2. *Clone* atau unduh repositori ini.
3. Buka terminal di dalam folder proyek.
4. Jalankan perintah instalasi dependensi:
   ```bash
   npm install
